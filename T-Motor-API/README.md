# Installation
```bash
pip install -e ".[mini]"       # core only — MotorBus, MITMotor, protocol
pip install -e ".[full]"       # + all motortools utilities re-exported via tmotorcan
pip install -e ".[gui]"        # PySide6 + pyqtgraph  (Motor-Monitor-GUI)
pip install -e ".[tui]"        # textual              (CAN-Analyzer-TUI, Model-Editor-TUI)
```
